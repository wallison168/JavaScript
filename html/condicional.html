Utilizando a Estrutura Condicional


Basicamente utilizamos a estrutura condicional ou estrutura de condição quando se tem a nescessidade de se trazer um resultado porem tendo que comprir com a condição definida ou seja basicamente definimos uma condição a ser aplicada no sistema, dependendo de qual condição for cumprida ele retornara um resultado em expecifico

Basicamente podemos definir uma estrutura condicional com 3 ou mais condições sendo assim a sua estrutura basica segue o seguinte parametro 

            
                let cor = color =>{
                    if(color === 'branco'){
                        return 'branco'
                    }
                    else if(color === 'amarelo'){
                        return 'amarelo'
                    }
                    else{
                        return 'cor indefinida'
                    }
                }
                
                console.log(cor('verde'))

Observa-se que o codigo definido acima foi passado para dentro de uma função uma estrutura condicional ou seja essa função recebeu um valor que foi passado pela estrutura condicional e por não comprido com nenhuma das condições definidas ele retornou o valor contido dentro do "else"

Basicamente em estruturas condicionais utilizamos 3 atributos para definir as condições.

1º if : basicamente nesse atributo definimos a primeira condição a ser aplicada

2º else if : basicamente utilizamos esse atributo quando se tem a nescessiade de se definir mais do que duas condições.

3º else : basicamente nesse atributo não e nescessario atribuir nenhuma condição ou seja nesse caso, caso o valor não cumpra com nenhuma das condições definida acima ele ira cair aqui no "else" sendo assim retornando o valor definido dentro dele

Utilizando operadores logicos

Tanto em estruturas condicionais com em laços de repetições são utilizados operadores logicos para fazerem comparações de um elemento com o outro

= : basicamente esse operador definir que um elemento ira receber o outro ou seja esta atribuindo o valor de um elemento a outro

== : basicamente esse operador compara se os valores são iguais ou seja nesse caso ele não leva em consideração o tipo do valor definido ou seja se o valor se trata de um valor do tipo "string" ou "numerico"

=== : basicamente esse operador e similar ao "==" ou seja e utilizado para comparar valores, o seu diferencial e que ele compara não so o valor mais tambem o tipo do valor.

!= : basicamente esse operador compara se um valor e diferente do outro

!== :basicamente esse operador e similar ao "!=" ou seja ele e utilizado para comparar se um valor e diferente de outro, o seu diferencial e que ele não compara somente o valor mas tambem o tipo do valor.

> :basicamente esse operador compara se um valor e maior que o outro

< :basicamente esse operador compara se um valor e menor que o outro

>= :basicamente esse operador compara se um valor e maior ou igual ao o outro

<= :basicamente esse operador compara se um valor e menor ou igual ao o

++ :basicamente esse operador incrementa o valor ou seja ele acrescenta (+1) no valor que recebeu esse operador

-- :basicamente esse operador decrementa o valor ou seja ele retira (-1) no valor que recebeu esse operador

|| :basicamente esse operador logico ("or" ou "||") e utilizado quando se tem a nescessidade de se realizar duas comparações dentro de uma condição ou seja nesse caso independente se uma das comparações forem cumpridas essa condição sera chamada.

&& :basicamente esse operador logico ("and" ou "&&") e similar ao operador "||" o seu diferencial e que no caso dele para que a condição seja chamada e nescessario que todas as comparações sejam verdadeiras.
